# 🍽️ Food Delivery — Landing Page

Landing page desenvolvida para um serviço de **delivery de refeições saudáveis**, com foco em experiência do usuário, clareza visual e conversão.

O projeto simula a apresentação de um produto real, destacando praticidade, qualidade e alimentação equilibrada.

---

## 🧠 Sobre o Projeto

Esta landing page foi criada para demonstrar a apresentação de um serviço de alimentação saudável, onde o usuário pode:
- Escolher refeições balanceadas
- Realizar pedidos com facilidade
- Receber comida fresca e quente

O layout foi pensado para ser simples, moderno e atrativo, priorizando a comunicação direta com o usuário final.

---

## ✨ Seções da Landing Page

- Hero section com chamada principal
- Apresentação do conceito de alimentação saudável
- Seção “Como funciona”
- Cards explicativos do processo
- Área de depoimentos
- Formulário de contato
- Call to action (CTA) estratégico

---

## 🎨 Destaques da Interface

- Design clean e moderno
- Uso estratégico de cores quentes
- Layout responsivo
- Componentes reutilizáveis
- Hierarquia visual clara
- Foco em conversão e experiência do usuário
